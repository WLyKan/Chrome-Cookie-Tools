# 插件测试清单

## 准备工作

### 1. 构建项目
```bash
npm run build
```
✅ 确认 `dist` 目录已生成且包含所有必要的文件

### 2. 加载插件
- 打开 `chrome://extensions/`
- 开启"开发者模式"
- 点击"加载已解压的扩展程序"
- 选择项目根目录
