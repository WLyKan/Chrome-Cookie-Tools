# Cookie Tools 测试清单

## 准备工作

### 1. 构建项目
```bash
pnpm build
```
✅ 确认 `.output/chrome-mv3` 目录已生成且包含所有必要的文件

### 2. 加载插件
- 打开 `chrome://extensions/`
- 开启"开发者模式"
- 点击"加载已解压的扩展程序"
- 选择项目的 `.output/chrome-mv3` 目录

## 功能测试

### 测试 1: 配置页面 - 保存配置（成功场景）

**测试步骤：**
1. 点击插件图标
2. 切换到"配置"标签页
3. 在"源网站URL"输入：`https://example.com`
4. 添加Cookie名称：`sessionId`, `token`, `userId`
5. 点击"保存配置"按钮

**预期结果：**
- ✅ 显示成功消息（绿色背景）
- ✅ 消息内容：`配置保存成功！`
